# 테라폼 (Terraform)
> `IaC(Infrstructure as a Code)` - 인프라 관리를 코드로 하는 개념

**설치 방법**
```terminal
brew install terraform
```

### 테라폼 버전 관리
특정 버전 혹은 여러 버전을 사용할 필요가 있을 때는 `tfenv`를 사용하면 편리하다.

```
brew install tfenv
```
